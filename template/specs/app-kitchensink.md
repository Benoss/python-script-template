# Kitchensink App

## Overview

The kitchensink app provides a single page that demonstrates the basic django-cotton components used across the project. It acts as a visual reference and usage guide for templates.

## Routes

- `/kitchensink/` renders the component showcase page.

## Components

The following components are defined under `templates/components/`:

- `button` (variants: primary, secondary, ghost, danger; sizes: sm, md, lg)
- `card` (title, subtitle, footer)
- `badge` (variants: neutral, info, success, warning, danger)
- `alert` (title, variants: info, success, warning, danger)
- `input` (label, help_text, error, placeholder, value)
- `table` (head/body slots, caption, density)

## Templates

- `django_starter/apps/kitchensink/templates/kitchensink/index.html` contains the examples and brief usage notes.
